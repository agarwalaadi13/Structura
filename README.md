## Deploy in 5 Minutes (Free, No Credit Card)

### Step 1 — Create GitHub Account
Go to https://github.com and create a free account if you don't have one.
Use a professional username (e.g. your name or initials).

### Step 2 — Create Repository
1. Click the green "New" button on GitHub
2. Repository name: `structura`
3. Set to **Public** (required for free GitHub Pages)
4. Click "Create repository"

### Step 3 — Upload Files
In your new repository:
1. Click "uploading an existing file"
2. Drag and drop ALL files from this folder:
   - `index.html`
   - `academy.html`
3. Click "Commit changes"

### Step 4 — Enable GitHub Pages
1. Go to repository Settings
2. Click "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Branch: `main`, Folder: `/ (root)`
5. Click Save

### Step 5 — Your site is live!
Wait 2-3 minutes. Visit:
`https://[your-username].github.io/structura/`

---

## Custom Domain (Optional — $12/year)

1. Buy a domain at Namecheap or Google Domains (e.g. `structurafinance.com`)
2. In GitHub Pages settings, enter your custom domain
3. At your domain registrar, add a CNAME record:
   - Name: `www`
   - Value: `[your-username].github.io`
4. Site will be live at `https://www.structurafinance.com`
