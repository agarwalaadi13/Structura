STRUCTURA — STRUCTURED CREDIT ANALYTICS PLATFORM

Structura is a full-stack structured products analytics platform covering the complete spectrum of fixed income credit — from Agency MBS and Non-Agency RMBS to CLOs, CMBS, ABS, and Whole Loans. The platform integrates live rate data from FRED and the Federal Reserve, a multi-tranche waterfall engine with OC/IC trigger mechanics, CPR/CDR cash flow modeling, OAS/duration analytics across 92 real CUSIPs, and a duration-based intraday live pricer that reprices securities off current coupon benchmark moves every 30 seconds.

The Whole Loan Trading Desk prices individual loans and pools using FICO/LTV/DTI inputs with real Fannie Mae LLPA grids, pulling live 30yr/15yr rates from FRED. The Pool Builder stratifies loan tapes by credit attribute, the Trade Blotter books best-efforts and mandatory deliveries with live SRP calculation, and the Strats Table provides WAC/FICO/LTV weighted-average stratification for due diligence. Rate data is sourced from FRED (St. Louis Fed) with hardened seeds as fallback — displayed as reference data, not real-time quotes.

STRUCTURA — TECH STACK
▸ Frontend: Pure HTML5/CSS3/ES6 — zero frameworks, zero build tools
▸ Charts: Chart.js 4.4.1 — canvas-based, real-time re-render
▸ Fonts: Space Mono (mono/data), Syne (headers), Inter (body)
▸ Market Data: FRED API (St. Louis Fed) — free, JSON, CORS-safe
▸ Backup Data: Hardcoded seeds (May 2026 actuals) — graceful fallback
▸ Cash Flow Engine: Custom JS — monthly amortization loop
▸ Waterfall: Sequential/Pro-Rata/Hybrid with OC/IC test logic
▸ IRR: Bisection method — 50-iteration convergence
▸ Sensitivity: 7×7 CPR/CDR matrix — 49 model runs per render
▸ Password: Client-side gate — dev mode reveals source logic
Total: ~3,200 lines HTML/CSS/JS | ~100KB | no backend required
Cash Flow Formula Reference
CORE MATH
SMM = 1 − (1 − CPR)^(1/12)
MDR = 1 − (1 − CDR)^(1/12)
PMT = Bal × [r(1+r)^n] / [(1+r)^n − 1]
Interest = Beg Bal × WAC/12
Defaults = Beg Bal × MDR
Prepay = (Beg Bal − Defaults) × SMM
Net CF = Int + Sched Prin + Prepay + Recovery
Recovery = Defaults × (1 − Severity)
PV(CF) = Net CF / (1 + disc/12)^m
Price = Σ PV(CF) / Original Balance × 100
WAL = Σ(Principal_t × t) / Σ Principal_t
Mac Dur = Σ(PV_t × t) / Σ PV_t
Mod Dur = Mac Dur / (1 + y/12)
Convexity = [P(+Δy)+P(−Δy)−2P] / (P×Δy²)
DV01 = Price × Mod Dur × 0.0001

Waterfall Logic Flow
SEQUENTIAL PAY WATERFALL
1. Collect total pool cash flow
2. Pay senior interest (A1, A2) pro-rata
3. Pay mezzanine interest (M, B1, B2)
4. OC/IC TEST: Is OC ≥ threshold?
↳ PASS → Release principal sequentially
↳ FAIL → Trap excess spread to OC
5. Pay A1 principal until zeroed
6. Pay A2 principal until zeroed
7. Pay M, B1, B2 in sequence
8. Allocate losses bottom-up (B2→B1→M)
9. OC tranche absorbs residual loss
Hybrid: switches to pro-rata when OC passes
